# Craftnet CLI

A CLI for the [Craftnet API](https://docs.api.craftcms.com/).

## Installation

To install the cli, run `composer global require superbig/craftnet-cli`

Make sure `~/.composer/vendor/bin/` is added to your PATH.

## Usage
*Show all the commands*
`craftnet`
